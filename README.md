# 🏪 Store Rating Platform

A full-stack Store Rating Platform built with the MERN stack.  
The application allows users to discover stores, submit ratings, and enables administrators and store owners to manage platform data through dedicated dashboards.

---

## 🚀 Live Features

### 🔐 Authentication & Authorization

- User registration
- Secure login with JWT authentication
- Password hashing using bcrypt
- Role-based access control

Supported roles:

- ADMIN
- USER
- OWNER

---

# 👨‍💼 Admin Panel

Admins have full control over the platform.

Features:

- View platform statistics
  - Total users
  - Total stores
  - Total ratings

- User management
  - Create Admin, User, and Store Owner accounts
  - View all users
  - Search users
  - Filter users by role
  - Sort users

- Store management
  - Create stores
  - Assign store owners
  - View all stores
  - Search and sort stores

---

## 👤 User Dashboard

Normal users can:

- Browse all available stores
- Search stores by name and address
- View average store ratings
- Submit a rating (1-5)
- Update previously submitted ratings
- Change account password

---

## 🏬 Store Owner Dashboard

Store owners can:

- View their store information
- View overall average rating
- See total number of ratings
- View users who rated their store
- Change account password
